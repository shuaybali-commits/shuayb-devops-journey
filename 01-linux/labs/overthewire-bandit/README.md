# OverTheWire Bandit

This section contains my solutions and notes for the OverTheWire Bandit wargame.

Bandit focuses on Linux fundamentals including:
- File navigation
- Permissions
- SSH usage
- Text processing
- Basic security concepts

Each level demonstrates a specific concept, with commands and explanations documented in each file.

## Structure

- level-00 → Introduction and SSH access
- level-01+ → Increasing difficulty with real-world Linux scenarios

## Skills Practiced

- Linux CLI navigation
- File permissions and ownership
- Searching and filtering data
- Working with encoded and hidden files
- Problem-solving in a Linux environment
