# Sad Servers

This section contains solutions and notes for Sad Servers scenarios.

Sad Servers simulates real-world Linux troubleshooting situations where systems are broken or misconfigured.

## What I Practiced

- Debugging system issues
- Identifying misconfigurations
- Fixing services and processes
- Investigating logs and system behaviour

Each scenario includes:
- Problem description
- Steps taken to investigate
- Commands used
- Final resolution
