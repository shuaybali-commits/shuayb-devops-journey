# Linux Basics

## Overview
Linux is a command-line based operating system used widely in servers and DevOps environments.

---

## Common Commands

### File & Directory Navigation

- `ls` – list files and directories  
  - `ls -l` → detailed view  
  - `ls -lh` → human-readable sizes  
  - `ls -t` → sort by time  
  - `ls -1` → one item per line  

- `pwd` – print current directory  

- `cd` – change directory  
  - `cd /` → root directory  
  - `cd ~` → home directory  
  - `cd ..` → move up one level  

---

### File & Directory Management

- `mkdir` – create directory  
- `rmdir` – remove empty directory  
- `rm` – remove files/directories  
  - `rm -r` → recursive  
  - `rm -f` → force  

- `touch` – create file  

---

### File Content & Searching

- `echo` – output text  
- `cat` – view or combine files  
- `grep` – search for specific text  

---

## Programs, Binaries, and Shell

- **Program** → instructions written by a developer  
- **Binary** → compiled version of a program  
- **Shell** → interprets commands and communicates with the system  

---

## Vim Basics

### Modes

## Key Concepts

### System Commands

- `head` / `tail` → view file content  
- `cp` / `mv` / `rm` → file operations  
- `sudo` / `su` → elevated privileges  

---

### Users & Permissions

- Users and groups control access  
- Permissions:
  - Read (r)  
  - Write (w)  
  - Execute (x)  

---

## Environment Variables

Environment variables store configuration values used by programs.

### Common Actions

- Set variables  
- Modify values  
- Apply to programs  
- Use during runtime
